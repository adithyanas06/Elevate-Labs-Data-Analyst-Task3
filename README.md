# Task 3 - SQL for Data Analysis

## Objective
The objective of this task is to learn how to manipulate and analyze structured data using SQL. The task involves writing SQL queries to retrieve, filter, sort, group, and analyze data from an e-commerce database.

## Tools Used
- DB Browser for SQLite
- SQLite Database
- GitHub

## Dataset
A sample E-commerce dataset containing the following tables:
- Customers
- Orders
- Products
- Order_Items

## SQL Concepts Covered
- SELECT
- WHERE
- ORDER BY
- GROUP BY
- Aggregate Functions (SUM, AVG, COUNT)
- INNER JOIN
- LEFT JOIN
- Subqueries
- Views
- Indexes

## Files Included
- ecommerce.db
- task3.sql
- customers.csv
- orders.csv
- products.csv
- order_items.csv
- Screenshots
- README.md

## Results
Successfully created an SQLite database, imported the dataset, executed SQL queries for data analysis, created a view, and optimized query performance using an index.

## Conclusion
This task helped me understand the fundamentals of SQL and database operations. I learned how to retrieve, filter, sort, join, and summarize data using SQL queries, as well as how to create views and indexes for efficient data analysis.
